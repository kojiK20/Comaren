6人以上コマ練に参加する際に自動で5コマ設定します。

################################################
1コマ目と2コマ目と4コマ目(少し変則的ですが希望がありましたので)はランクの近い人同士が組むようになってます。

3コマ目は参加者全体の中でランクが半分くらい離れてる人同士が組みやすくなってます。

5コマ目はランクの遠い人同士が組むようになってます。

多球のメンバに関してはランダムに3人組みます。多球につく回数が偏らないようになってます。
################################################

外部の方は上位にしたかったら↑を、下位にしたかったら↓をつけてください。何もなければ中位になります。

生成する度に違う結果が返るので気に入らなかったり不具合があったら組み直してください。

入力しやすいようにスペースは全角を認識して、半角だと上手くいかないので注意。(ただし途中参加/抜けの+-は半角。ややこしい)

使用可能な台数を指定できます。10台なら全て多球でMax30人までいけます。9台なら27人です。

部員かどうかは同一ディレクトリ内のmembers.txt内の名前で判断してるので同性の方等は注意してください。

部内ランクもmembers.txtを参照するので部内ランクの変動の度にファイルを更新してください。
